📋 检查规则说明

基本规范

  • 使用全角中文标点符号，句末要有句号
  • 数学公式应使用 LaTeX 格式
  • 中文与英文、数字或公式之间以半角空格隔开

题解内容

  • 不应出现大量无关内容（闲聊、吐槽、求赞等）
  • 不应完整复制题面
  • 必须包含主要思路和算法分析
  • 不应使用生成式AI辅助产生关键内容

排版规范

  • 使用 #, ##, ### 表示标题
  • 使用 -, +, * 表示无序列表
  • 使用反引号表示行内代码
  • 使用 ``` 表示代码块

LaTeX规范

  • 数学公式应写在 $...$ 或 $$...$$ 中
  • 公式中的文本应使用 \text{}
  • 上下标应使用 _ 和 ^
  • 位运算应使用 \operatorname{and}, \operatorname{or} 等
